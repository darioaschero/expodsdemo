How to use this page

Set · core · active scale · …

Switch the Neutral selector in the topbar to mark a scale as active. The active scale is the one --nuri-color-neutral-* aliases in tokens-primitive.css point at — so the whole docs site (and any component using semantic tokens) re-themes accordingly. Toggle the theme switch to compare each scale in light and dark. Watch how each scale's step 9–10 reads against the lilac brand in Colour · Semantic (where accent and neutral compose live).

Functionally the candidate scales are interchangeable (same Radix 12-step role model). The choice is aesthetic: which neutral best supports the brand without competing with it.

Candidate scales

Each scale exposes 24 tokens (12 steps × 2 themes). Pick one consistent active scale via the topbar; the others stay addressable via their explicit name (--nuri-color-gray-N-light, etc.) until lock-in.
