Approach

The type system lives entirely at the primitive layer: six numeric steps, no role names. xs, sm, md, lg, xl, and 3xl, with a deliberate gap reserved at 2xl for a future intermediate between xl (30) and 3xl (57).

Each step has two states — a default regular weight and an emphasized weight that's always semibold (600) across all leaves (operator-locked at N+6.5 polish · previously bold for the large range, simplified to one weight). Tracking tightens at larger sizes to keep optical density consistent.

Token shape

Each step exposes five atomic CSS variables under --nuri-type-{step}: -size, -line-height, -tracking, -weight (default), and -em-weight (emphasized).

Line-height is a unitless ratio — CSS multiplies it against the resolved font-size at render. Same convention React Native uses (you compute lineHeight = size × ratio at the call site). Each step's ratio is hand-picked to keep optical density consistent as size grows — tighter at the top of the scale (1.19 at 3xl), looser at the bottom (1.38 at xs).

Tracking is in em so it rescales proportionally too. The RN sync pipeline multiplies em × size at export to emit a dp letterSpacing. Utility classes (.nuri-type-md, .nuri-type-md--em, …) bundle everything for CSS consumers.

Font family is out of scope Type steps deliberately don't set font-family. The active family is determined per platform — system stack in production, swapped via the Font toggle in the topbar for previewing iOS, Android, or Pixel.

Roadmap

  • planned
    Per-component typography aliases

    E.g. --nuri-button-label-type, --nuri-card-title-type — alias the primitive steps inside each component's own token file. Until then, components reference --nuri-type-{step} directly.

  • planned
    Fill the 2xl gap

    Promote the reserved slot to a real step (likely 40 or 44) when a screen needs an intermediate between xl (30) and 3xl (57).

  • planned
    Lock the font-family source of truth

    Today the platform toggle (ios / android / pixel) swaps the body font in typography.css. Decide whether this lives at the primitive level, the platform-set level, or both.
