API

The props signature. web ↔ RN is 1:1 at the prop names + accepted values; the web side dispatches via attribute-selector CSS, the RN side reads the value and slots it into a View style object (see docs/migration-tests/button-matrix/index.tsx for the reference RN implementation).

Prop Type Default Description
direction "column" | "row" "column" Flex axis. column stacks vertically; row lays out horizontally.
gap "xs" | "sm" | "md" | "lg" | "xl" unset Inter-child spacing. Maps to --nuri-space-{xs..xl} · 5-leaf subset of the 7-leaf spacing vocabulary (2xs and 2xl stay available ad-hoc via direct tokens.space.*).
align "start" | "center" | "end" | "stretch" | "baseline" unset Cross-axis alignment (flex align-items). Browser default applies if unset.
justify "start" | "center" | "end" | "between" | "around" unset Main-axis alignment (flex justify-content).
wrap boolean false Whether overflowing children wrap to the next line (flex flex-wrap). Meaningful primarily on direction="row".
fill boolean false Grow to fill the main axis of a flex parent (flex: 1 0 auto) — e.g. a <nuri-box fill> inside a Scroll, giving an inner Spacer the slack to push trailing content to the end (decision 60).
as string "div" Host element on the web side ("section", "nav", …). Has no RN analogue — RN always renders <View>.

Anatomy & parts

Stack is a single-part component. The web side renders an inner <div> (or the element named by as) inside the display:contents wrapper; the RN side renders a <View>.

Token mapping

Prop value → CSS property → DTCG → CSS var → reference. Only gap dereferences a semantic token; the other props dispatch to CSS literals (flex enum values). Layout primitives have no @layer tokens block per decision 37 — the dispatch happens entirely in @layer rules through attribute selectors (.nuri-stack[data-gap="md"] { gap: var(--nuri-space-md); }).

Roadmap

  • shipped · N+11
    fill prop

    Promoted in N+11 (decision 60) so a Stack can fill a Box fill inside a Scroll, giving an inner Spacer room to push trailing content to the bottom. RN map: a View with style={{ flex: 1 }}.

  • planned
    Divider prop + <nuri-divider> component

    External-project spec ships a divider boolean on Stack that wraps a thin separator between children. Deferred to N+6.3+ pending real consumer. When promoted, lands as Stack-internal child-wrapping in JS + RN render-map · requires shipping <nuri-divider> too.

  • planned
    RN spec long-term home

    Today the Stack RN definition lives in the migration-test pair as a local module. Long-term: peer file lib/components/stack/stack.tsx? Separate lib/spec/ namespace? Re-evaluate post-N+6.2 when n≥2 component RN specs exist.

  • planned
    2xs / 2xl gap leaves

    N+6.2 ships gap as a 5-leaf subset (xs/sm/md/lg/xl) of the 7-leaf semantic space vocabulary. The 2xs (2px) and 2xl (36px) leaves stay available ad-hoc via direct tokens.space.{2xs,2xl}. Promote to the prop surface if a real consumer surfaces the need.
